## ES原理

### 1、概念

ES是一个分布式，基于Restful风格的数据搜索和分析引擎。

### 2、倒排索引

倒排索引表中每一项都包括一个属性值和具有该属性值对应记录的地址。

由于不是按照记录来确定属性值，而是由属性值来确定记录的位置，因此被称为倒排索引。

### 2. ES集群底层（健康状态、节点类型、crud具体流程）

健康状态：

-   绿色：主从分片一致。
-   黄色：主分片一致，副分片未同步。
-   红色：主分片未同步。

节点类型：

- master node
- Coordinating node
- data node
- ingest node
